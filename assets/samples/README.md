# 样本图片目录

本目录用于存放《杖剑传说》地图定位器的样本图片。

## 准备说明

1. **截图要求**
   - 使用游戏内截图工具或系统截图工具
   - 确保定位器清晰可见
   - 包含周围的地图元素作为参考

2. **图片格式**
   - 支持 PNG、JPG 格式
   - 推荐 PNG 格式（保留透明度）

3. **命名规范**
   - 使用英文命名，如 `target_1.png`、`marker_corner.png`
   - 避免使用中文字符和特殊符号

4. **样本数量**
   - 建议：5-10 张
   - 涵盖不同位置、不同角度的定位器
   - 可以包括不同类型的定位器（传送点、任务点等）

## 使用方式

将样本图片直接放入此目录，Agent 会自动加载。

例如：
```
assets/samples/
├── locator_1.png
├── locator_2.png
├── quest_marker.png
└── teleport_point.png
```

## 注意事项

1. 确保图片清晰度足够
2. 避免图片过大（建议不超过 200KB）
3. 定期更新样本以适配游戏更新
4. 如果识别准确率偏低，可在此目录添加更多样本
